# PCORS

> Simple **P**roxy that adds **CORS** headers

## Usage

```txt
pcors <port> <target>

Run CORS Proxy

Positionals:
  port    Local listen port                                             [number]
  target  Target URL                                                    [string]

Options:
  --help     Show help                                                 [boolean]
  --version  Show version number                                       [boolean]

Not enough non-option arguments: got 0, need at least 2
```
